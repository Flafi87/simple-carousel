# Simple Multipurpose Carousel

The component accepts an array of elements in props slidesArray and a settings Object
`<Carousel slidesArray={posts} settings={settings} />`

Default settings:
    width = "100%",
    height = "400px",
    arrow = true,
    dots = true,
    backgroundColor = "none",
    arrowColor = "black",
    autoplay = false,
    autoplaySpeed = 2000,
    neverend = false,
